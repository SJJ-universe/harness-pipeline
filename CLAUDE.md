# Skills Workspace

이 폴더는 73개 커뮤니티 스킬이 설치된 Claude Code 워크스페이스입니다.

## 설치된 스킬 카테고리

- **Superpowers (14)**: brainstorming, writing-plans, executing-plans, TDD, debugging 등
- **Harness (1)**: 에이전트 팀 자동 설계 메타스킬
- **Engineering (20)**: agent-designer, database, docker, CI/CD, performance 등
- **Toolkit (23)**: python, react, nextjs, typescript, security 등
- **Product/Marketing (8)**: content, copywriting, landing-page, research 등
- **Others (7)**: senior-backend/frontend/fullstack, playwright 등

## 스킬 사용법

대화 중 관련 작업을 요청하면 자동으로 적합한 스킬이 활성화됩니다.
